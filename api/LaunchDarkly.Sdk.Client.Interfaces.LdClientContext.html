Encapsulates SDK client context when creating components.

Remarks

Factory interfaces like IDataSourceFactory receive this class as a parameter. Its public properties provide information about the SDK configuration and environment. The SDK may also include non-public properties that are relevant only when creating one of the built-in component types and are not accessible to custom components.

Some properties are in BasicConfiguration instead because they are required in situations where the LdClientContext has not been fully constructed yet.
