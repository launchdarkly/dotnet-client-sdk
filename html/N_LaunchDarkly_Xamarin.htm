LaunchDarkly.Xamarin Namespace

This is the main namespace for the LaunchDarkly Xamarin SDK. You will most often use LdClient (the SDK client) and Configuration (configuration options for the client). The SDK also uses types from LaunchDarkly.Client, such as User.

Convenience methods that extend the ILdClient interface.
LdClient
A client for the LaunchDarkly API. Client instances are thread-safe. Your application should instantiate a single LdClient for the lifetime of their application.
