Remarks

This is equivalent to SetOfflineAsync(Boolean), but as a synchronous method.

If you set the property to , any existing connection will be dropped, and the method immediately returns .

If you set it to when it was previously , but no connection can be made because the network is not available, the method immediately returns , but the SDK will attempt to connect later if the network becomes available.

If you set it to when it was previously , and the network is available, the SDK will attempt to connect to LaunchDarkly. If the connection succeeds within the interval maxWaitTime, the method returns . If the connection permanently fails (e.g. if the mobile key is invalid), the method returns . If the connection attempt is still in progress after maxWaitTime elapses, the method returns , but the connection might succeed later.
