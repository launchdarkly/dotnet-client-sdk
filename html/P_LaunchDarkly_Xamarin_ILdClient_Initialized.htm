Remarks

When you first start the client, once Init(Configuration, User, TimeSpan) or InitAsync(Configuration, User) has returned, Initialized should be if and only if either 1. it connected to LaunchDarkly and successfully retrieved flags, or 2. it started in offline mode so there's no need to connect to LaunchDarkly. If the client timed out trying to connect to LD, then Initialized is (even if we do have cached flags). If the client connected and got a 401 error, Initialized is . This serves the purpose of letting the app know that there was a problem of some kind.

If you call Identify(User, TimeSpan) or IdentifyAsync(User), Initialized will become until the SDK receives the new user's flags.
