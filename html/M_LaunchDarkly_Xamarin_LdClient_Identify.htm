Remarks

This is equivalent to IdentifyAsync(User), but as a synchronous method.

If the SDK is online, Identify(User, TimeSpan) waits to receive feature flag values for the new user from LaunchDarkly. If it receives the new flag values before maxWaitTime has elapsed, it returns . If the timeout elapses, it returns (although the SDK might still receive the flag values later). If we do not need to request flags from LaunchDarkly because we are in offline mode, it returns .

If you do not want to wait, you can either set maxWaitTime to zero or call IdentifyAsync(User).
